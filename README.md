### Quick Start
Windows _cmd_:

1. install [Node.js](https://nodejs.org/en/download/)
Then check
```
node --version
npm --version
```

2. Clone the repo and go to the folder
```
git clone https://github.com/alekseyIsakin/app
cd app
```

3. Install Express
```
npm install express
```

4. Run Node.js applicationcd
```
node app.js
```

Go to _http://127.0.0.1:3000_ in your browser
